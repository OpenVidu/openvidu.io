# OpenVidu Single Node COMMUNITY installation

OpenVidu Single Node COMMUNITY is a production-ready deployment option that provides all the necessary features for running real-time applications with medium user load.

Install OpenVidu Single Node COMMUNITY in your preferred environment:

- [**On-premises installation**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/on-premises/install/index.md): set up on your own servers.
- [**AWS installation**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/aws/install/index.md): deploy to Amazon Web Services.
- [**Azure installation**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/azure/install/index.md): deploy to Microsoft Azure.
- [**Google Cloud Platform installation**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/gcp/install/index.md): deploy to Google Cloud Platform.

Once your deployment is complete, refer to the following sections for configuration and management:

- [**On-premises: configuration and administration**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/on-premises/admin/index.md)
- [**AWS: configuration and administration**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/aws/admin/index.md)
- [**Azure: configuration and administration**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/azure/admin/index.md)
- [**Google Cloud Platform: configuration and administration**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/gcp/admin/index.md)

If you want to upgrade your OpenVidu Single Node COMMUNITY installation, refer to this section:

- [**On-premises: upgrade OpenVidu Single Node Community**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/on-premises/upgrade/index.md)
- [**AWS: upgrade OpenVidu Single Node Community**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/aws/upgrade/index.md)
- [**Azure: upgrade OpenVidu Single Node Community**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/azure/upgrade/index.md)
- [**Google Cloud Platform: upgrade OpenVidu Single Node Community**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/gcp/upgrade/index.md)

We also offer installation tutorials for other Cloud Providers:

- [**Oracle Cloud Infrastructure**](https://openvidu.io/3.5.0/docs/self-hosting/single-node/oracle/install-tutorial/index.md)
