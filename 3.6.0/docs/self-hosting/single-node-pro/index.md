# OpenVidu Single Node PRO

OpenVidu Single Node PRO brings all the features of OpenVidu Single Node COMMUNITY plus [**2x performance**](https://openvidu.io/3.6.0/docs/self-hosting/production-ready/performance/index.md) and [**advanced observability**](https://openvidu.io/3.6.0/docs/self-hosting/production-ready/observability/index.md). It is aimed to users that want a single-node setup but still want OpenVidu PRO features.

Info

Scalability and fault tolerance require a multi-node setup. If you need them, consider using [OpenVidu Elastic](https://openvidu.io/3.6.0/docs/self-hosting/elastic/index.md) or [OpenVidu High Availability](https://openvidu.io/3.6.0/docs/self-hosting/ha/index.md).
