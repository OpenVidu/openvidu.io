# E2E Encryption

OpenVidu Meet supports **end-to-end encryption (E2EE)** for audio, video, and chat. When enabled, only the participants in the meeting can decrypt the media — not even the OpenVidu Meet server can access the content.

E2EE can be enabled on a per-room basis when [creating](https://openvidu.io/3.7.0/meet/features/rooms/creation-management/#create-rooms) or [editing a room](https://openvidu.io/3.7.0/meet/features/rooms/creation-management/#edit-rooms).
