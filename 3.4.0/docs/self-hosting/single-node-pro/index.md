# OpenVidu Single Node PRO installation

OpenVidu Single Node PRO brings all the features of OpenVidu Single Node COMMUNITY plus [**2x performance**](https://openvidu.io/3.4.0/docs/self-hosting/production-ready/performance/index.md) and [**advanced observability**](https://openvidu.io/3.4.0/docs/self-hosting/production-ready/observability/index.md). It is aimed to users that want a single-node setup but still want OpenVidu PRO features.

Info

Scalability and fault tolerance require a multi-node setup. If you need them, consider using [OpenVidu Elastic](https://openvidu.io/3.4.0/docs/self-hosting/elastic/index.md) or [OpenVidu High Availability](https://openvidu.io/3.4.0/docs/self-hosting/ha/index.md).

Install OpenVidu Single Node PRO in your preferred environment:

- [**On-premises installation**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/on-premises/install/index.md): set up on your own servers.
- [**AWS installation**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/aws/install/index.md): deploy to Amazon Web Services.
- [**Azure installation**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/azure/install/index.md): deploy to Microsoft Azure.

Once your deployment is complete, refer to the following sections for configuration and management:

- [**On-premises: configuration and administration**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/on-premises/admin/index.md)
- [**AWS: configuration and administration**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/aws/admin/index.md)
- [**Azure: configuration and administration**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/azure/admin/index.md)

If you want to upgrade your OpenVidu Single Node PRO installation, refer to this section:

- [**On-premises: upgrade OpenVidu Single Node PRO**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/on-premises/upgrade/index.md)
- [**AWS: upgrade OpenVidu Single Node PRO**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/aws/upgrade/index.md)
- [**Azure: upgrade OpenVidu Single Node PRO**](https://openvidu.io/3.4.0/docs/self-hosting/single-node-pro/azure/upgrade/index.md)
