# OpenVidu Elastic installation

OpenVidu Elastic is part of the PRO edition of OpenVidu. You have the following deployment options:

- [**On-premises installation**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/on-premises/install/index.md): Set up OpenVidu Elastic on your own servers.
- [**AWS installation**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/aws/install/index.md): Deploy OpenVidu Elastic on Amazon Web Services.
- [**Azure installation**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/azure/install/index.md): Deploy OpenVidu Elastic on Microsoft Azure.

Once your deployment is complete, refer to the following sections for configuration and management:

- [**On-premises: configuration and administration**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/on-premises/admin/index.md)
- [**AWS: configuration and administration**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/aws/admin/index.md)
- [**Azure: configuration and administration**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/azure/admin/index.md)

If you want to upgrade your OpenVidu Elastic installation, refer to this section:

- [**Upgrade OpenVidu Elastic On-premises**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/on-premises/upgrade/index.md)
- [**Upgrade OpenVidu Elastic AWS**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/aws/upgrade/index.md)
- [**Upgrade OpenVidu Elastic Azure**](https://openvidu.io/3.4.1/docs/self-hosting/elastic/azure/upgrade/index.md)
