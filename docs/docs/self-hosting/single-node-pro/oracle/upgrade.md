# Upgrade OpenVidu Single Node <span class="openvidu-tag openvidu-pro-tag" style="font-size: .6em; vertical-align: text-bottom">PRO</span>: Oracle Cloud Infrastructure

--8<-- "shared/self-hosting/oracle-provider-chip.md"

In Oracle Cloud Infrastructure, we recommend upgrading by redeploying [OpenVidu Single Node PRO Oracle Cloud Infrastructure](../oracle/install.md) with the latest version.

However, if you prefer to upgrade the OpenVidu Single Node PRO without redeploying, you can follow the steps outlined in the [Upgrade OpenVidu Single Node PRO - On Premises](../on-premises/upgrade.md) section.
